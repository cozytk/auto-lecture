# sync-report-w3

- 대상 파일: `.claude/skills/slide-master/SKILL.md`
- 변경 목적: 슬라이드 본문과 발표자 스크립트 간 싱크 누락을 줄이고, 최신성 기준과 자동 검증 기준을 명문화

## 반영 내용

- 핵심 원칙에 제목 형식 규칙 추가: 개념/섹션 제목은 `1. {제목}` 사용, `개념 1: {제목}` 금지
- `최신성 규칙` 섹션 추가: 현재 연도 기준 최신 모델/날짜 사용, `GPT-4`·`2024` 같은 구식 기준을 최신 사례처럼 쓰지 않도록 명시
- `자가 검증` 아래에 `스크립트-슬라이드 싱크 검증` 추가:
  - 모든 `v-click` 단계에 대응하는 `[클릭]` 필요
  - 스크립트에서 언급한 시각 요소가 실제 슬라이드에 존재해야 함
  - `시간: X분` 대비 1분당 약 200자 기준으로 스크립트 길이 점검
- `Playwright 검증` 섹션 추가: 렌더링 후 자동 시각 검증을 권장하고, 오버플로·겹침·단계 공개 오류·누락 요소·노트 불일치를 점검하도록 명시

## 검증 메모

- 기존 발표자 노트 규칙의 "1분당 약 200자" 기준과 새 싱크 검증 규칙이 충돌하지 않도록 같은 기준으로 정리함
- 자가 검증 영역에 새 검증 항목을 배치해 생성 후 체크리스트 흐름을 유지함
