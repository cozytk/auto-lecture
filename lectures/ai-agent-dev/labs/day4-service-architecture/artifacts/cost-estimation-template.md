## 참고: 비용 계산 예시

### 예시: 일 500건 요청, 모델 티어링 적용

```
[Simple: 40% = 200건, gpt-4o-mini]
  200 x 2회 x 1000 tokens(in) x $0.15/1M = $0.06
  200 x 2회 x 500 tokens(out) x $0.60/1M = $0.12
  소계: $0.18/일

[Standard: 35% = 175건, gpt-4o-mini]
  175 x 3회 x 2000 tokens(in) x $0.15/1M = $0.16
  175 x 3회 x 800 tokens(out) x $0.60/1M = $0.25
  소계: $0.41/일

[Complex: 25% = 125건, gpt-4o]
  125 x 4회 x 3000 tokens(in) x $2.50/1M = $3.75
  125 x 4회 x 1000 tokens(out) x $10.00/1M = $5.00
  소계: $8.75/일

일일 합계: $9.34
월간 합계: $280 (30일)
캐싱 25% 적용 후: $210/월
```
