### 서비스 아키텍처 실습

Agent 서비스를 프로덕션에 올리기 위한 핵심 패턴을 직접 구현하고 비교한다.

### 학습 목표

- Dev/Staging/Prod 환경 분리를 YAML 기반 설정 로더로 구현한다
- 복잡도 기반 모델 라우터로 비용을 최적화한다
- 임베딩 유사도 기반 Semantic Cache를 구현한다
- LangGraph로 Multi-Agent 파이프라인을 구성한다
- 토큰 기반 비용 산정 및 예산 알림을 설정한다

### 실습 구성

| 단계 | 핵심 질문 | 구현 내용 |
|------|-----------|-----------|
| 환경 분리 | Dev와 Prod가 왜 달라야 하나? | YAML 기반 설정 로더 (AppConfig) |
| 모델 라우터 | 모든 요청에 비싼 모델을 써야 하나? | 복잡도 기반 자동 라우팅 |
| Semantic Cache | 비슷한 질문마다 API를 호출해야 하나? | 임베딩 유사도 캐시 |
| Multi-Agent | 하나의 Agent가 모든 걸 해야 하나? | LangGraph 파이프라인 |
| 비용 산정 | 월 비용이 얼마나 나올까? | 토큰 기반 비용 계산기 |

### 사전 준비

- OpenAI API Key
- Langfuse 환경 (트레이스 기록용)
