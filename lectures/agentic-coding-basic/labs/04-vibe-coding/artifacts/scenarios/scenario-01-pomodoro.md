# 시나리오 1: CLI 포모도로 타이머

**난이도**: ★☆☆ (입문 추천)
**예상 완성 시간**: 30~40분
**언어**: Python (표준 라이브러리만 사용)

---

## 앱 개요

포모도로 기법을 CLI에서 사용할 수 있는 타이머 앱.
25분 집중 → 5분 휴식을 반복하며 작업 세션을 관리한다.

---

## 필수 기능 (30점 만점 기준)

### 기능 1: 타이머 실행
- `python pomodoro.py start` — 포모도로 시작 (25분 카운트다운)
- `python pomodoro.py break` — 휴식 타이머 시작 (5분)
- 카운트다운 중 남은 시간을 터미널에 표시 (1초마다 갱신)
- 타이머 종료 시 터미널 벨 소리 또는 시각적 알림

### 기능 2: 세션 기록
- 완료된 포모도로 세션을 `sessions.json`에 저장
- 저장 항목: 시작 시각, 종료 시각, 타입(집중/휴식), 완료 여부

### 기능 3: 통계 조회
- `python pomodoro.py stats` — 오늘의 완료 세션 수 출력
- 총 집중 시간(분) 출력
- 연속 포모도로 개수 출력

---

## 선택 기능 (도전 정신 15점)

- `python pomodoro.py stats --week` — 주간 통계
- 작업명 입력: `python pomodoro.py start --task "보고서 작성"`
- 진행률 바 표시: `[=========>         ] 18:32 남음`
- 설정 파일(`config.json`)으로 집중/휴식 시간 커스터마이즈

---

## 테스트 요구사항

- 타이머를 실제 25분 기다리지 않고 짧은 시간(5초)으로 테스트할 수 있어야 함
- `pytest` 또는 `unittest`로 아래 항목 검증:
  - 세션 저장/불러오기
  - 통계 계산 정확성
  - 타이머 종료 조건

## 프로젝트 구조 예시

```

```
